-----Install Dependencies-----
1) Open project folder
2) open cmd by clicking the path and typing cmd and hit enter
3) Type "pip install -r requirements.txt" and wait for the installation to complete


-----Run the Project-----
1) open cmd in the same path and type cmd then hit enter
2) type command "python app.py"
3) Type the address URL  generated in the browser
